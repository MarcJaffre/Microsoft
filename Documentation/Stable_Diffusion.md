## webui-user.bat (AMD)
Permet de rendre accessible Stable Diffusion sur Internet
```
@echo off
set PYTHON=
set GIT=
set VENV_DIR=
set COMMANDLINE_ARGS=--precision full --no-half --use-cpu all --skip-torch-cuda-test --listen --port 80
call webui.bat
```
